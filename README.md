Movie Website(moviear) :
A modern and responsive movie discovery website built using React
It fetches live movie data from The Movie Database(TMDB) API, featuring trending and popular films with a clean UI.

Key Features :

-  Browse Trending & Popular Movies (Real-time data)
-  Search movies by title
-  Movie details: rating, overview, release date, cast details
-  Recommendation to similar movies based on user 
-  Dark theme UI
-  Fully responsive design for all screens
-  Deployed on Netlify
